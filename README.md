## Catalog Test

**О проекте:** Тестовое задание - SPA с каталогом (с фильтрацией) и страницей товара. Реализованы основные сценарии работы с товарами: удаление, редактирование, создание, добавление в избранное.


**Стек:** 
- Typescript;
- React NextJS;
- Redux Toolkit;
- React-Hook-Form;
- Axios;
- TailwindCSS;
- React Transition Group

Установка зависимостей:

```bash
npm install
```

Запуск:

```bash
npm run dev
# or
yarn dev
```
